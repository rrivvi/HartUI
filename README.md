An unofficial but faithful continuation of the now-archived WinForms UI library [`CuoreUI.Winforms`](https://github.com/owtysm2/CuoreUI).

> [!TIP]
> Don't be afraid to report problems or suggest new controls and components on the [![GitHub](https://i.imgur.com/rJ9l299.png) Issues](https://github.com/rrivvi/CuoreUI-ng/issues) page!

---

## 🔧 Installation

### ![NuGet logo](https://i.imgur.com/6aPyVAg.png) NuGet

The official NuGet package is [**CuoreUI-ng.Winforms** ](https://www.nuget.org/packages/CuoreUI-ng.Winforms).

> [!TIP]
> Package Manager Console:
>
> ```powershell
> Install-Package CuoreUI-ng.Winforms
> ```

### 📦 Add to the Toolbox

Follow the step-by-step guide:

> ![GitHub logo](https://i.imgur.com/rJ9l299.png) [Load CuoreUI-ng.dll from the Toolbox](./.github/load-into-toolbox.md)
